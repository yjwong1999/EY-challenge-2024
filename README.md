# EY-challenge-2024

## Setup environment
- Clone the repo
```bash
git clone https://github.com/yjwong1999/EY-challenge-2024.git
cd streaming
```

- Create conda environment
```bash
conda create --name ey-challenge python=3.8.10 -y
conda activate ey-challenge
```
